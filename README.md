# MolinoSeguro

Proyecto desarrollado para la materia Seminario de Práctica de Informática – Universidad Siglo 21.

El sistema MolinoSeguro permite gestionar incidentes laborales mediante la utilización de Programación Orientada a Objetos, persistencia con MySQL y la aplicación de los patrones DAO y MVC.

## Tecnologías utilizadas

- Java
- MySQL
- JDBC
- IntelliJ IDEA
- XAMPP
- GitHub

## Funcionalidades implementadas

- Registro de incidentes
- Consulta de incidentes
- Actualización de estados
- Generación de alertas
- Generación de reportes
- Persistencia mediante MySQL
- Arquitectura MVC
- Patrón DAO
